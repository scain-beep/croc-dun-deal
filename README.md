# Crocodile Dun-Deal — Minimal Starter (Fixed)

Zero-scroll hero with two CTAs + compact “How It Works”. App Router + TS.

## Quick Start
```bash
npm i
npm run dev
# open http://localhost:3000
```

### What’s included
- `/app/page.tsx` – no-scroll homepage
- `/app/start/page.tsx` – wizard stub
- `/app/meet-shaun/page.tsx` – intro video stub
- `/app/api/lead/route.ts` – server route (stub)
- `/app/globals.css` – global tokens
- `/app/Home.module.css` – component styles
